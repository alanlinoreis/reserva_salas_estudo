# Documentação do uso de IA no projeto

Este diretório documenta todas as interações relevantes com IA (Claude, da Anthropic)
durante o desenvolvimento do Sistema de Reservas de Salas de Estudo.

Cada arquivo desta pasta corresponde a uma etapa do desenvolvimento e contém:

- **Prompt utilizado**: a pergunta ou pedido enviado à IA.
- **Resposta resumida**: o que a IA retornou (resumo, não cópia literal).
- **Decisão tomada**: o que eu, como desenvolvedor, optei por aplicar ou descartar.
- **Adaptações realizadas**: ajustes que fiz no código sugerido.
